# Hoard things you know how to do
- URL: https://simonwillison.net/guides/agentic-engineering-patterns/hoard-things-you-know-how-to-do/
- Added: 2026-03-02 08:00:51
- Tags: #agent #advice

## TL;DR
囤积已解决问题的知识与代码，结合 AI 编码助手，通过重新组合创造新解决方案，大幅提高开发效率。

## Summary
核心主题：囤积你知道如何做的事情，这是智能体工程模式中的一种有效策略，能够提高与 AI 编码助手的工作效率。

关键信息：

1.  囤积知识的重要性
   - 软件开发技能的一部分是理解什么可能实现，什么不可能实现
   - 知道理论上的可能性与亲见其实现有区别
   - 深度收集解决方案，最好配有可运行的代码示例

2.  囤积方式
   - 博客和 TIL (Today I Learned) 博客记录已解决的问题
   - GitHub 仓库收集项目代码，特别是概念验证代码
   - 使用 LLM 扩充解决方案集合
   - tools.simonwillison.net 收集 HTML 工具和原型
   - simonw/research 仓库存放更复杂的示例

3.  重新组合囤积物
   - 囤积物成为编码助手的有力输入
   - 提示模式：让助手结合两个或多个现有工作示例构建新东西
   - 案例：结合 Tesseract.js 和 PDF.js 创建浏览器端 OCR 工具

4.  编码助手增强囤积价值
   - 只需有用技巧一次，助手可参考文档示例解决未来相似项目
   - 编码助手擅长搜索，可查找本地或网络上的示例
   - 可指示助手克隆仓库并使用其中代码作为输入

# Building a High-End AI Desktop

- Added: 2026-03-18 01:45:55

## TL;DR
作者以 7,500 欧元购入企业级 Grace-Hopper 系统，经改造后成功在家庭环境运行 235B 参数 AI 模型，总成本约 9,000 欧元，远低于市场价。

## Summary
核心主题：本文讲述了一位爱好者如何以 7,500 欧元的价格购入企业级 Grace-Hopper 超级芯片系统，并将其改造为可在家庭环境中运行 235B 参数 AI 模型的高性能桌面电脑的过程。

关键信息与逻辑脉络：

1.  **背景与动机**
    -   本地运行大型语言模型通常需要在昂贵消费级 GPU 和企业级硬件之间做出妥协
    -   作者发现 Reddit 上有人以 7,500 欧元的价格出售本应价值超过 100,000 欧元的 Grace-Hopper 系统，决定冒险购买

2.  **硬件规格与交易**
    -   系统配置：2x Grace-Hopper 超级芯片、2x 72 核 Grace CPU、2x H100 GPU、总计 1152GB 高速内存、900 GB/s NVLink-C2C 带宽
    -   该系统原为液冷服务器，已被改装为风冷，卖家通过在线商店销售改装后的服务器设备
    -   作者实地考察并完成交易，系统虽布满灰尘但能正常启动

3.  **改造过程**
    -   **清洁与准备**：使用异丙醇彻底清洁主板上积累的灰尘
    -   **水冷系统改造**：购买 Arctic Liquid Freezer III 420 AIO 水冷器，使用 3D 打印和 CNC 加工定制适配器板
    -   **结构组装**：使用 ProfilAlu 铝型材构建框架，设计并打印数十个定制支架

4.  **遇到的问题与解决方案**
    -   **风扇连接问题**：尝试改装风扇连接器导致"噼啪"声和冒烟，估算电流需求有误
    -   **风扇驱动故障**：多个风扇驱动器失效，通过禁用 BMC 风扇检查服务解决
    -   **温度传感器异常**：GPU 温度传感器显示 1,677,7214°C（接近 24 位无符号整数最大值），表明传感器连接松动或损坏
    -   **微修复**：在显微镜下修复损坏的表面贴装元件，包括 100nF 电容和 4.7k 电阻

5.  **最终配置与性能**
    -   完成系统组装，安装 8TB E1.S NVMe SSD 和 3kW 48V 电源
    -   解决 GPU 驱动问题，禁用 NVLink 以允许 GPU 独立初始化
    -   性能测试结果：能够流畅运行 235B 参数模型，如 gpt-oss-120b-Q4_K_M、GLM-4.5-Air-Q4_K_M 和 Qwen3-235B-A22B-Instruct-2507-Q4_K

6.  **成本分析**
    -   总成本约 9,000 欧元，远低于单块 H100 GPU 的市场价格
    -   主要开支包括 Grace-Hopper 服务器（7,500 欧元）、定制水冷适配器（700 欧元）和存储设备（250 欧元）

7.  **结论**
    -   成功将价值 80,000 美元的企业级硬件改造为家用 AI 桌面，能够运行 235B 参数模型
    -   过程充满挑战，包括调试异常温度传感器和显微镜下焊接元件，但最终成果值得
    -   作者强调，这种改造需要专业知识、耐心和一定的冒险精神

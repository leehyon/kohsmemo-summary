## Summary
**核心主题**
本文介绍了提升 OpenClaw AI Agent 实用性的 10 大必装技能，涵盖搜索增强、内容处理及效率提升三个维度，旨在帮助用户将工具从“能用”升级为“好用”。

**关键信息与逻辑脉络**

*   **搜索类技能（连接真实世界）**
    *   **tavily-search**：专为 AI 设计的搜索引擎，返回结构化结果，支持网页搜索、内容提取、深度爬取及站点映射，需配置 `TAVILY_API_KEY`。
    *   **multi-search-engine**：整合 17 个搜索引擎，覆盖国内外平台（如百度、Google、DuckDuckGo），最大优势是无需 API Key，并支持高级搜索语法。

*   **内容处理技能（从混乱到有序）**
    *   **summarize**：一键摘要网页、PDF 及 YouTube 视频，支持调整摘要长度，提升阅读效率。
    *   **markdown-converter**：将 PDF、Word、PPT、图片（OCR）、音频等多格式文档统一转换为 Markdown，便于 LLM 处理。
    *   **news-aggregator-skill**：聚合 8 大信息源（如 Hacker News、GitHub Trending、36Kr），支持关键词过滤并生成简报。
    *   **rss-subscribe**：订阅任意 RSS 源，自动追踪博客及新闻网站更新。

*   **效率工具技能（干掉重复劳动）**
    *   **openclaw-github-assistant**：通过自然语言在聊天中操作 GitHub，支持列仓库、查 CI、创建 Issue 等。
    *   **email-sender-tw**：邮件发送工具，支持模板、附件及批量发送，适用于自动化通知场景。
    *   **docker**：提供 Docker 操作指南，涵盖镜像构建、编排、网络及安全加固的最佳实践。
    *   **humanizer-zh**：去除 AI 写作痕迹，修复夸张、宣传性语言及高频词汇，使文字更自然。

*   **安装方式**
    *   使用 `npx clawhub@latest install <skill-name>` 或 `skillhub install <skill-name>` 进行安装。
